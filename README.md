Combing the components:

MPU6050	| ESP32-C6
-------------------
VCC	    | 3.3V
GND	    | GND
SCL     |	GPIO20
SDA     |	GPIO19

The Grove Base has extra pins to connect the Pkcell LP503562 to it.
The ESP32C6 fits on top of the Base without needing any soldering.
